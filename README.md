# Agent Yuki Security Dashboard - YUKI

A Next.js application for YUKI security platform that helps defend crypto assets through AI-powered smart contract security scanning.

## Features

- **Security Scanning**: AI-powered analysis of smart contracts
- **Community Dashboard**: View security incidents and vulnerability reports
- **Multi-chain Support**: Support for Ethereum, BNB Chain, Solana, and other blockchain networks
- **Real-time Analytics**: Track security metrics and fund protection statistics

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

```bash
npm run build
npm start
```
